#Erikbot V 1.0 - Von Erik Fietz

Erstellt im Juni 2012 für den Kurs "Linuxprogrammierung" der FH Heidelberg.

**Inhalt:** Quellcode, GIT-Log, Markdown-Readme, Manpage, Makefile
